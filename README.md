# truffle-template
Simple template for rocking truffle environment 


```
yarn install
```

### create a .env
```
ETHERSCAN_API_KEY=
INFURA_API_KEY=

PRIVATE_KEY=
TEST_PRIVATE_KEY=
DEV_ADDRESS=
```

### Configuration
Configure `addresses.js`


### Verify
```
truffle run verify Fun@0x00 --network main
```
